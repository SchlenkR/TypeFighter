JS emitter

Golden tests for TypeFighter → JavaScript string emission. The emitter doesn't depend on the solver — it's a pure AST walker — so these tests construct expressions directly via the X.* DSL.

literal number

Inferred
F# test body 
jsOf (X.Lit 42) |> shouldEqual "42"

match with literal arms and wildcard

Inferred
F# test body 
let expr =
    X.Fun (X.Ident "n") (
        X.Match (X.Var "n") [
            X.Arm (X.PatLit 0) (X.Lit "zero")
            X.Arm (X.PatLit 1) (X.Lit "one")
            X.Arm X.PatWild (X.Lit "other")
        ])
let expected =
    "(n) => ((__s) => { if (__s === 0) return \"zero\"; "
    + "if (__s === 1) return \"one\"; return \"other\"; })(n)"
jsOf expr |> shouldEqual expected

match with var pattern binds scrutinee

Inferred
F# test body 
let expr =
    X.Match (X.Lit 42) [
        X.Arm (X.PatVar "x") (X.Var "x")
    ]
jsOf expr |> shouldEqual "((__s) => { return ((x) => x)(__s); })(42)"

toJsModule wires imports and exports default

Inferred
F# test body 
let expr = X.App (X.Var "log") (X.Lit "hi")
let result =
    Emit.toJsModule
        [ "log", "(s) => console.log(s)" ]
        expr
let expected = "const log = (s) => console.log(s);\nexport default log(\"hi\");\n"
result |> shouldEqual expected
